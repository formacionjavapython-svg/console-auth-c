# Console Auth Lab

Un proyecto de laboratorio para experimentar con autenticación en aplicaciones de consola.

## Descripción

Este proyecto es un entorno de prueba para implementar y probar diferentes métodos de autenticación en aplicaciones de línea de comandos (CLI).

## Estructura del Proyecto

```
console-auth-lab/
├── src/
│   ├── main/
│   │   └── java/          # Código fuente principal
│   └── test/
│       └── java/          # Pruebas unitarias
├── scripts/               # Scripts de utilidad
├── README.md             # Este archivo
└── .gitignore            # Archivos ignorados por Git
```
